# 🚀 ChipForce API - Backend

Sistema completo de atendimento WhatsApp com múltiplos chips.

## 🛠️ Tecnologias

- **Node.js** + Express
- **Socket.IO** - Comunicação em tempo real
- **UUID** - Identificadores únicos
- **Helmet** - Segurança
- **CORS** - Cross-origin
- **Morgan** - Logs

## 🚀 Instalação

```bash
npm install
npm start
